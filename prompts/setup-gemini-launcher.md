# Gemini CLI 실행 설정 및 런처 생성 기록

본 문서는 윈도우 환경에서 Gemini CLI를 효율적으로 실행하기 위해 진행된 설정 및 파일 생성 기록을 담고 있습니다.

## 1. 요청 사항 (User Request)
- 윈도우 사용자 환경에서 원하는 경로에서 Gemini CLI를 실행할 수 있는 방법 마련.
- 특정 경로(`bgpt/web/html`)에서 즉시 실행 가능한 배치 파일(`gemini-launcher.bat`) 생성.
- 위 내용들에 대한 상세한 사용 가이드(HTML) 요청.

## 2. 생성된 솔루션 및 파일 목록

### A. 우클릭 메뉴 등록 스크립트 (`setup_context_menu.bat`)
- **역할**: 윈도우 파일 탐색기의 마우스 우클릭 메뉴에 "여기에서 Gemini CLI 열기" 항목을 추가합니다.
- **특징**: 관리자 권한으로 1회 실행하면 어느 폴더에서든 Gemini CLI를 해당 경로에서 즉시 열 수 있습니다.

### B. 전용 런처 배치 파일 (`gemini-launcher.bat`)
- **역할**: 더블 클릭 시 `C:\Users\PC\Desktop\bgpt\web\html` 경로로 자동 이동 후 Gemini CLI를 실행합니다.
- **특징**: 특정 프로젝트 폴더 전용 실행 파일로, 바탕화면에 바로가기를 만들어 사용하기에 최적화되어 있습니다.

### C. 사용자 가이드 문서 (`gemini_usage_guide.html`)
- **역할**: 위 두 파일의 차이점과 상세한 사용 방법을 시각적으로 설명합니다.
- **내용**: 단계별 실행 방법, 관리자 권한 실행 주의사항, 활용 팁 등 포함.

## 3. 실행 방법 요약
1. **즉시 실행**: `gemini-launcher.bat` 파일을 더블 클릭합니다.
2. **시스템 설정**: `setup_context_menu.bat` 파일을 마우스 우클릭하여 **'관리자 권한으로 실행'**합니다. 이후 어느 폴더에서나 마우스 우클릭으로 실행이 가능해집니다.
