Gemini CLI 슬래시 명령어 가이드

Gemini CLI에서 사용할 수 있는 주요 슬래시 명령어 목록입니다.

일반 및 세션 관리

명령어 설명
/about버전 정보를 표시합니다.
/help (또는 /?)도움말 정보를 표시합니다.
/quit (또는 /exit)Gemini CLI를 종료합니다.
/chat (또는 /resume)세션 및 체크포인트를 관리합니다.
/clear터미널 화면과 세션 기록을 지웁니다.
/stats세션, 모델, 도구의 사용 통계를 표시합니다.

설정 및 외관

명령어 설명
/settings대화형 설정 편집기를 엽니다.
/theme시각적 테마를 변경합니다.
/editor지원되는 에디터를 선택합니다.
/terminal-setup터미널 키바인딩을 구성합니다.
/vimVim 입력 모드를 토글합니다.

워크플로우 및 작업 공간 제어

명령어 설명
/directory (또는 /dir)작업 공간 디렉토리를 관리합니다.
/plan계획 모드(Plan Mode)로 전환하거나 현재 계획을 봅니다.
/rewind대화 기록을 뒤로 돌려 이전 상태로 복구합니다.
/restore파일을 특정 도구 실행 전 상태로 복구합니다.
/init현재 디렉토리를 분석하여 GEMINI.md 파일을 생성합니다.
/memoryGEMINI.md 파일의 컨텍스트를 관리합니다.
/shells백그라운드 쉘 보기를 토글합니다.

확장 및 에이전트 관리

명령어 설명
/agents로컬 및 원격 서브 에이전트를 관리합니다.
/extensionsGemini CLI 확장을 관리합니다.
/skills에이전트 스킬을 관리합니다.
/mcpModel Context Protocol(MCP) 서버를 관리합니다.

모델 및 성능

명령어 설명
/model사용할 모델 구성을 관리합니다.
/compress토큰 절약을 위해 대화 내용을 요약본으로 대체합니다.
/tools사용 가능한 도구 목록을 표시합니다.

유틸리티 및 지원

명령어 설명
/auth인증 방법을 변경합니다.
/bug버그 리포트를 작성합니다.
/copy마지막 출력을 클립보드에 복사합니다.
/docs브라우저에서 공식 문서를 엽니다.
/upgradeGemini Code Assist 업그레이드 페이지로 이동합니다.
