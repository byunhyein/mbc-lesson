```html 블럭 레벨 vs 인라인 레벨

🧱 블럭 레벨(Block Level) 과 ✏️ 인라인 레벨(Inline Level)

🎈 먼저 쉽게 생각해보자!

HTML 태그는 크게 2가지 친구가 있어요!

  • 🧱 블럭 레벨 태그 = 혼자 한 줄을 다 차지하는 큰 상자
  • ✏️ 인라인 레벨 태그 = 글자처럼 옆으로 붙어있는 작은 친구
🍱 도시락 칸처럼 생각하면 쉬워요!
블럭은 "큰 칸", 인라인은 "작은 반찬" 느낌!

🧱 블럭 레벨(Block Level)

블럭 태그는 혼자 한 줄을 모두 사용해요!

div 태그
p 태그
h1 태그

✅ 특징

  • 📏 가로를 끝까지 차지해요
  • 🆕 자동으로 줄바꿈이 돼요
  • 📦 큰 상자처럼 사용해요
  • 🎨 width, height 설정이 쉬워요

📚 대표적인 블럭 태그 종류

태그 설명
<div> 영역 나누기
<p> 문단
<h1> ~ <h6> 제목
<ul> 목록
<li> 목록 아이템
<section> 구역 나누기

✏️ 인라인 레벨(Inline Level)

인라인 태그는 글자처럼 옆으로 붙어요!

span a strong

✅ 특징

  • ➡️ 옆으로 나란히 있어요
  • 📏 필요한 크기만 사용해요
  • 📝 글자 꾸밀 때 많이 써요
  • 🚫 width, height 적용이 어려워요

📚 대표적인 인라인 태그 종류

태그 설명
<span> 글자 영역
<a> 링크
<strong> 강조
<em> 기울임 강조
<img> 이미지

🆚 블럭 vs 인라인 한눈에 보기

비교 블럭 레벨 인라인 레벨
줄바꿈 ⭕ 자동 줄바꿈 ❌ 줄바꿈 없음
크기 한 줄 전체 사용 필요한 만큼만 사용
느낌 📦 큰 상자 ✏️ 글자
대표 태그 div, p, h1 span, a, strong

🎯 아주 중요한 한마디!

🧱 블럭 = "혼자 한 줄!"
✏️ 인라인 = "옆으로 붙기!"

😄 이것만 기억해도 HTML 절반은 이해한 거예요!

```
