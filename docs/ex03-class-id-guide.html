🏷️ CSS 클래스 vs 아이디

이름표와 교복! 누구인지 알려주는 마법의 꼬리표

🆔 아이디 (ID Selector)

🪪

아이디는 '주민등록번호'나 '이름표'와 같아요!
세상에 딱 한 명만 가질 수 있는 특별한 이름이에요.

  • ☝️ 딱 한 번만! 한 페이지에 같은 아이디는 하나만 써야 해요.
  • #️⃣ CSS에서는 이름 앞에 샵(#)을 붙여서 불러요.
  • 🎯 특정 요소 딱 하나만 콕 집어서 꾸밀 때 사용해요.
/* CSS에서 부를 때 */ #my-name { color: blue; } <p id="my-name">나는 특별해!</p>

👕 클래스 (Class Selector)

👕

클래스는 '교복'이나 '팀 복'과 같아요!
여러 명의 친구가 똑같은 옷을 입을 수 있는 이름이에요.

  • 🙌 여러 번! 똑같은 클래스 이름을 여러 태그에 줄 수 있어요.
  • .️⃣ CSS에서는 이름 앞에 점(.)을 붙여서 불러요.
  • 🎨 비슷한 친구들을 한꺼번에 똑같이 꾸밀 때 사용해요.
/* CSS에서 부를 때 */ .red-team { color: red; } <p class="red-team">1번 선수</p> <p class="red-team">2번 선수</p>

🆚 한눈에 비교해볼까?

구분 🆔 아이디 (ID) 👕 클래스 (Class)
기호 # (샵) . (점)
사용 횟수 딱 한 번만! (유일함) 여러 번 가능! (재사용)
비유 이름표, 주민등록번호 교복, 팀 유니폼
언제 써요? 특별한 딱 하나를 꾸밀 때 여러 개를 한꺼번에 꾸밀 때
💡 선생님의 팁!
한 친구가 여러 개의 교복을 입을 수도 있어요!
class="blue-team thick-text" 처럼 클래스는 공백을 두고 여러 개를 쓸 수 있지만,
아이디는 무조건 하나만 가져야 한다는 걸 꼭 기억해!

🌟 클래스와 아이디만 잘 써도 CSS 박사가 될 수 있어요!
