💌 마법 우체국 💌

안녕! 여기는 컴퓨터에게 편지를 보내는 마법 우체국이야.
HTML 폼(Form)은 우리가 쓴 편지를 컴퓨터에게 잘 전달해주는 봉투 같은 역할을 해!

<input type="text"> 한 줄 글쓰기 칸

이름이나 별명처럼 짧은 글을 쓸 때 사용해.

<input type="password"> 비밀번호 마법 칸

비밀번호를 쓰면 글자가 ●●● 모양으로 변해서 아무도 못 보게 해줘!

쉿! 이건 우리만의 비밀이야.

<input type="radio"> 딱 하나만 고르기

여러 개 중에서 딱 하나만 고를 수 있는 동그란 버튼이야.

가장 좋아하는 계절은?

<input type="checkbox"> 여러 개 고르기

먹고 싶은 과일을 모두 고르는 것처럼 여러 개를 체크할 수 있어!

오늘 먹고 싶은 간식은? (모두 골라봐!)

<select> 상자에서 고르기

상자를 누르면 목록이 짠! 하고 나타나서 고를 수 있어.

<button> 편지 보내기!

이제 모든 내용을 다 썼다면 우체통에 넣어야겠지?
