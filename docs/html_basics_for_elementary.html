🏠 초등학생을 위한 HTML 기본 가이드

웹 페이지는 하나의 집과 같아요!

📘 1. HTML이 뭔가요?

HTML은 웹 페이지를 만드는 언어예요. 우리가 책을 쓸 때 한글을 쓰듯이, 컴퓨터는 HTML이라는 언어로 웹 페이지를 "써요".

🧠 집에 비유해서 이해해봐요!

HTML로 만드는 웹 페이지는 한 채의 집이라고 생각하면 돼요.

HTML 부분 집의 어디? 설명
<!DOCTYPE html> "이 집은 현대식 집이에요!" 간판 브라우저(크롬, 사파리)에게 "이건 최신 HTML5 문서야"라고 알려주는 거예요
<html> 집 전체 땅 위에 있는 집 전체를 뜻해요. 모든 것을 다 감싸는 가장 큰 울타리예요
<head> 서재 / 창고 손님이 보지 않지만, 집을 운영하는 데 필요한 정보가 있는 곳. 주소, 설계도 같은 것들!
<body> 거실, 방, 주방 손님이 와서 실제로 보고 사용하는 공간. 여기에 사진, 글자, 버튼을 놓는 거예요

🗺️ 2. HTML 구조도 보기

기본 구조도

<!DOCTYPE html> ← "이 집은 최신식 집입니다!" (간판) <html> ← 🏠 집 전체 │ ┌───┴───┐ ▼ ▼ ┌─────┐ ┌─────┐ │head │ │body │ ← 서재/창고 거실+방들 │서재 │ │거실 │ └──┬──┘ └──┬──┘ │ │ ▼ ▼ ┌────┐ ┌────────┐ │제목│ │ 제목 │ ← <h1> │설정│ │ 글 │ ← <p> │그림│ │ 사진 │ ← <img> │주소│ │ 버튼 │ ← <button> └────┘ └────────┘

<head> 안에 자주 들어가는 태그

<head> ├── <meta charset="UTF-8"> ← 문자 인코딩 (한글 안 깨지게) ├── <meta name="viewport" ...> ← 반응형 설정 (핸드폰에서도 예쁘게) ├── <title>제목</title> ← 브라우저 탭 제목 ├── <link rel="stylesheet" ...> ← 외부 CSS 연결 (옷 입히기) ├── <script src="..."></script> ← 외부 JS 연결 (움직이게 하기) └── <style> ... </style> ← 내부 CSS

<body> 안의 시맨틱 구조 (권장 레이아웃)

<body> ├── <header> ← 로고, 검색창 등 │ └── <nav> ← 메뉴, 네비게이션 │ ├── <main> ← 페이지의 핵심 콘텐츠 │ ├── <section> ← 주제별 구분 │ │ └── <article> ← 독립적인 콘텐츠 │ └── <aside> ← 사이드바, 부가 정보 │ └── <footer> ← 저작권, 연락처 등

✏️ 3. 내 첫 번째 HTML 코드

아래 코드를 복사해서 메모장에 붙여넣고, "index.html"이라는 이름으로 저장한 뒤 브라우저로 열어보세요!

<!-- 이 문서는 HTML5입니다 --> <!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <title>내 첫 웹 페이지</title> </head> <body> <h1>안녕! 나는 민수야</h1> <p>나는 초등학생이고 HTML을 배우고 있어.</p> <p>내 취미는 축구야! ⚽</p> </body> </html>

🧩 4. 각 태그가 뭔지 쉽게 설명

  • <html> = 집 전체 (모든 것을 감싸는 큰 상자)
  • <head> = 서재 (사람들 눈에 안 보이지만 중요한 정보)
    • <title> = 집 이름표 (브라우저 탭에 써지는 이름)
  • <body> = 거실과 방 (사람들이 실제로 보는 곳)
    • <h1> = 큰 제목 (방문 앞에 붙이는 현관판)
    • <p> = 문단, 글 (방 안에 써 놓은 편지)
💡 꼭 기억하세요!

  1. 모든 것은 <html> 안에 있어야 해요 — 집 밖에 가구를 놓을 수 없잖아요?
  2. <head>는 보이지 않는 정보, <body>는 보이는 내용 — 서재와 거실의 차이!
  3. 태그는 열고(<>) 닫아(</>)야 해요 — 문을 열고 나가면 다시 닫아야 하잖아요?

📚 5. 더 알아보기

HTML5에서는 시맨틱 태그를 사용해서 의미 있는 구조를 만드는 것이 권장돼요.

태그 의미 집에 비유하면
<header> 머리말, 소개 현관 (집의 첫인상)
<nav> 메뉴, 네비게이션 집 안의 안내판
<main> 페이지의 핵심 내용 거실 (가장 중요한 공간)
<section> 주제별 구분 방 (침실, 서재 등)
<article> 독립적인 글 방 안의 일기장
<aside> 부가 정보 베란다, 창고
<footer> 꼬리말, 정보 집 뒤편의 안내판
