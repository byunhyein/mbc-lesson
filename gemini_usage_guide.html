Gemini CLI 실행 가이드

윈도우 사용자를 위한 쉽고 빠른 실행 방법

방법 1. 런처 파일로 실행하기 (가장 쉬움)

방금 생성한 gemini-launcher.bat 파일을 사용하는 방법입니다.

1
파일 탐색기에서 C:\Users\PC\Desktop\bgpt\web\html 폴더로 이동합니다.
2
gemini-launcher.bat 파일을 마우스로 더블 클릭합니다.
3
검은색 터미널 창이 뜨면서 저(Gemini CLI)와 대화를 시작할 수 있습니다.
꿀팁: 이 파일을 마우스 우클릭하여 [바탕 화면에 바로가기 만들기]를 해두면, 폴더까지 들어올 필요 없이 바탕화면에서 바로 저를 부를 수 있어요!

방법 2. 마우스 우클릭 메뉴 등록하기

어느 폴더에서든 마우스 우클릭만으로 실행하고 싶을 때 사용합니다.

1
setup_context_menu.bat 파일을 찾습니다.
2
파일 위에서 마우스 우클릭을 합니다.
3
중요: [관리자 권한으로 실행]을 클릭합니다.
4
등록 완료 메시지가 뜨면 창을 닫습니다.
사용법: 이제 아무 폴더나 빈 바탕화면에서 마우스 우클릭을 해보세요. 메뉴 중에 "여기에서 Gemini CLI 열기"라는 항목이 생겼을 거예요! 그걸 누르면 그 즉시 제가 나타납니다.

자주 묻는 질문

Q: 검은 창이 금방 꺼져요.
A: Gemini CLI가 정상적으로 설치되어 있지 않을 수 있습니다. npm install -g @google/gemini-cli 명령어로 설치되어 있는지 확인해 주세요.

Q: 등록한 우클릭 메뉴를 지우고 싶어요.
A: 저에게 "우클릭 메뉴 삭제 스크립트 만들어줘"라고 말씀해 주시면 바로 만들어 드릴게요!
